前置了解： 配置文件分为两种

  1. 网站配置文件 (主配置文件)： _config.yml
  2. 主题配置文件： themes/主题名/_config.yml
  3. 所有配置都遵循 yaml 语法

使用 Butterfly 主题需要安装的插件汇总：

  • 渲染插件：hexo-renderer-pug、hexo-renderer-stylus 必备
  • 音乐插件： hexo-tag-aplayer
  • 豆瓣插件：hexo-douban

其他主题推荐：

  1. NexT：是一个高质量并且优雅的 Hexo 主题。使用人数多 (旧版 15.4k start、新版 6k start)，教程 / 解决方案多
  2. Volantis: 一个高度模块化和可定制化、功能相对完善的 Hexo 博客主题，既可以用作个人博客，也适合用来搭建文档。

格式：

Unique name： 当前多标签在当前整篇文章中唯一标识 (不可以带逗号)

  • 它是所有子标签的前缀 (如果没有指定子标题的名称的话)
  • 如果名称中有空格将被 “-” 代替
  • 在当前文章中唯一，不可以重复

[index]：默认展示的标签页。可选参数

  • 如果没有指定，默认选择第一个选项卡 (1)
  • 如果索引为 -1，则不会选择任何选项卡

[Tab caption]：当前标签的标题。可选参数

  • 如果没有指定标题，则使用 Unique name + 标签索引 作为标题
  • 如果没有指定标题，但指定了 icon，就显示图标

[@icon]：FontAwesome 图标名 (例如：‘fas fa-font’)，标签名和图标名之间可以加空格也可以不加

3.5：流程图外挂 (mermaid)

mermaid 标签不允许嵌套于一些隐藏属性的标签外挂，例如: tag-hide 内的标签外挂和 tabs 标签外挂，这会导致 mermaid 图示无法正常显示，使用时请留意。

请不要压缩 html 代码，不然会导致 mermaid 显示异常

使用 mermaid 标签可以绘制 Flowchart（流程图）、Sequence diagram（时序图 ）、Class Diagram（类别图）、State Diagram（状态图）、Gantt（甘特图）和 Pie Chart（圆形图），具体可以查看 mermaid 文档

3.6：标签隐藏外挂 (tag-hide)

如果你想把一些文字、内容隐藏起来，并提供按钮让用户点击显示。可以使用这个标签外挂。

不要在隐藏标签内使用 h1~h6 的标题，不然会影响 Toc

butterfly 支持三种标签隐藏外挂方式：

inline 在文本里面添加按钮隐藏内容，只限文字

content 不能包含英文逗号，可用 &sbquo; 代替

格式：

content：文本内容

display：按钮显示的文字 (可选)

bg：按钮的背景颜色 (可选)

color：按钮文字的颜色 (可选)

3.7：链接按钮外挂 (Button)

就是对链接跳转的一些美化

格式：

[url]：链接

[text]：按钮文字

[icon]： [可选] 图标

[color]：[可选]，按钮背景颜色

  • 默认 style
  • 按钮字体和边框颜色 ( outline 时)
  • default / blue / pink / red / purple / orange / green

[style]：[可选] 按钮样式

  • 实心 (默认)
  • outline
  • 留空

[layout]：[可选] 按钮布局

  • line (默认)
  • block
  • 留空

[position]：[可选] 按钮位置，前提是设置了 layout 为 block 默认为左边

  • center
  • right
  • 留空

[size]：[可选] 按钮大小

  • larger
  • 留空
